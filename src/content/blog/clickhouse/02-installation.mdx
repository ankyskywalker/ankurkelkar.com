<p class="lead text-xl text-gray-300 mb-8">
    Ready to start querying? Let's get ClickHouse installed. We'll use
    Docker for the quickest setup, but you can also install it natively
    on Linux or macOS.
</p>

<h2>Using Docker</h2>
<p>
    The easiest way to try ClickHouse is via Docker. Run the following
    command:
</p>
<pre class="language-bash">
docker run -d --name clickhouse-server --ulimit nofile=262144:262144 -p 8123:8123 -p 9000:9000 clickhouse/clickhouse-server
</pre>

<h2>Connecting to the Client</h2>
<p>
    Once the container is running, you can connect using the
    command-line client:
</p>
<pre class="language-bash">
docker exec -it clickhouse-server clickhouse-client
</pre>
<p>
    You should see a prompt like <code>:)</code>. This is the friendly
    ClickHouse smiley!
</p>

<h2>Your First Query</h2>
<p>Let's verify everything is working:</p>
<pre class="language-sql">
SELECT 'Hello, ClickHouse!'
</pre>

<h2>Conclusion</h2>
<p>
    That was easy! You now have a running ClickHouse server. In the next
    part, we'll dive into the most important engine in ClickHouse:
    MergeTree.
</p>
